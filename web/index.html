📖 使用说明

🎯 这是什么

一台街机风格的视觉质量评估打分台。 拖入图片或视频，系统自动计算画质分数并给出评级（S / A / B / C / D）。

📋 两种模式

FR 全参考：需要原始参考素材和待测素材两份文件。 算法逐像素 / 逐结构比对差异，回答"相比原图损失了多少"。

NR 无参考：只需待测素材一份文件。 算法直接分析画面自然度或深度学习特征，回答"这张图/这段视频本身好不好"。

🧪 七种算法

PSNR FR
峰值信噪比。逐像素算误差，单位 dB，越大越好（≥45 极好）。
SSIM FR
结构相似度。亮度/对比度/结构三通道对比，0~1，越大越好。
VMAF FR
深度学习全参考。ResNet50 双分支 + 特征差异回归，0~100 分，越大越好。
VMAF-V FR 仅视频
视频级深度学习全参考。ResNet18 + 8 帧时序池化，0~100 分，越大越好。
NIQE-Lite NR
轻量无参考。自然统计特征偏离分析，~3~30 分，越小越好。
VSFA NR
深度学习无参考。VGG16 逐帧推理，输出 DMOS，越小越好。
VSFA-V NR 仅视频
视频级深度学习。4 帧时序采样 + VGG16 聚合，MOS 越大越好。仅视频。

🕹 怎么操作

1. 首页按 Enter 或点"开始游戏"进入选择页
2. 选 FR/NR 模式 → 选算法 → 设置参数 → 点"开打！"
3. 拖入/点击上传参考图和测试图（NR 仅需测试图）
4. 点"开始评估" → 倒计时 → 实时进度 → 自动跳结算页
5. 结算页可"再来一次"（保留数据）、导出 JSON 报告

⌨ 快捷键

Enter / Space：标题页 → 选择页；结算页 → 再来一次
Esc：任意页面返回标题页
